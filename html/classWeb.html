Detailed Description

Represents a web formed by one or more live ranges.

A web groups together compatible live ranges belonging to the same variable. It stores all lines where the variable is alive and provides utility methods for interference analysis.

◆ addLiveRange()

void Web::addLiveRange ( LiveRange *  lr)

Adds a live range to the web.

The live range pointer is stored internally and all its live lines are inserted into the web line set.

If the web variable name has not yet been defined, it is initialized using the variable associated with the first inserted live range.

Time Complexity:

  • O(n log m)

Where:

  • n is the number of lines in the live range.
  • m is the number of lines already stored in the set.

Space Complexity:

  • O(n)
Parameters
lrPointer to the live range to add.

◆ interfereWith()

bool Web::interfereWith ( const Web &  other) const

Determines whether this web interferes with another web.

Two webs interfere if they share at least one active line, except for the special case where one web ends exactly when the other begins.

The method computes the intersection between both line sets and analyzes overlapping live ranges.

Time Complexity:

  • O(n + m + k * (a + b))

Where:

  • n is the number of lines in this web.
  • m is the number of lines in the other web.
  • k is the size of the intersection.
  • a and b are the number of live ranges in each web.

In practice, the dominant operation is the set intersection.

Space Complexity:

  • O(k)

Where:

  • k is the size of the intersection.
Parameters
otherWeb to compare against.
Returns
true if the webs interfere, false otherwise.
